Why is sketching a user interface a good idea?

Sketching the user interface (UI) before development can be very helpful, especially for complex applications like an electric scooter rental app. It allows your team and stakeholders to quickly understand the potential look and function of the interface.

How to Choose a Tool for Sketching UI

There are many tools available for sketching user interfaces. The right choice depends on your experience with visualizing ideas and arranging elements. We will use Figma and MockFlow as examples in this guide.

Prerequisites

Before you start, create an account on Figma and MockFlow. This guide uses a requirement specification translated from the Kravspecification Svenska Elsparkcyklar AB.

Sketching UI Using MockFlow

Sketch Navigation Menu

  1. Add an iOS Frame:
    • Select Elements > Frames > iOS Frame Light.
    • Set its dimensions to 390 x 844.
  2. Add Text Labels:
    • Drag Elements > Text > Label onto the canvas and modify it.
    • Add icons if desired.
  3. Align and Adjust Labels: Arrange the labels with icons for a functional navigation layout.

Sketch Customer View Over Trip History

  1. Add Text and Currency Labels: Display trip date and cost.
  2. Enhance Design: Add a horizontal line for visual separation.

Sketching UI Using Figma

Sketch Navigation Menu

  1. Set Up Frame:
    • Use Frame > Phone > iPhone 13 & 14 dimensions 390 x 844.
  2. Add Text: Add buttons with labels like “Map,” “Scan,” and “Profile.”
  3. Align Icons and Text: Arrange icons beside text, and add Auto Layout for consistent spacing.

Sketch Customer View Over Trip History

  1. Add Text and Icons: Include trip details and action icons.
  2. Adjust Sizes and Layout: Use Auto Layout for structured spacing between trip items.
  3. Refine the Design: Add borders and spacing for a polished look.
